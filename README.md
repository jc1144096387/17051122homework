# 项目验收
## 页面结构与css布局方式
页面分为固定的头部导航栏，采用两列布局的主体部分和侧边栏，以及旁边的悬浮侧边栏和折叠侧边栏

两列布局使用浮动实现       
旁边的悬浮侧边栏和折叠侧边栏采用绝对定位position: fixed;

## 悬浮广告框
这是我自制的jquery插件，从代码设计方面来说，这是我比较满意的部分         
插件化使其能够更好的重用，在参数方面，我遵循可配置和约定优于配置的原则，设置了1个必选项和多个可选项。     
具体介绍见jquery-floatBanner.js文件的注释

## 轮播图
轮播图没有插件化，需要html和css的配合   
在这里我大致说一下js的实现思路    
轮播图大致可以分为两个功能模块
- 鼠标事件交互
- 自动轮播
我们为右边的每个小图片和滑块背景都添加上事件监听。 当鼠标移入小图片时，将对应的大图片的透明度和z-index设为1，将其他大图片的透明度设为0，z-index设为-2，并实现滑块背景的移动 当鼠标移出时，调用auto()函数启动自动轮播   

更具体的介绍以及分离出来的代码，请看https://github.com/jc1144096387/banner 

## 蜘蛛网动态背景
这是我根据别人网站压缩后的代码修改后得到的，并添加了较为详细的注释。         
这个背景采用canvas实现，比较巧妙的是他直接利用js创建canvas元素，通过获得窗口宽高来设置canvas元素的宽高，并提供了一些参数供使用者配置，使用者只需要在页面中引入 <script type="text/javascript" src="assets/js/canvas-nest.js" zIndex="-10"></script> 即可，而不需要再添加canvas标签，同样遵循约定优于配置的原则，没有设置参数时便会采用设定好的默认值。       
将小方块抽象为一个对象，随机生成位置，并存入数组中，其中还生成了一个鼠标所在位置的小方块，便于之后实现鼠标交互效果，之后遍历数组，判断各个方块间的距离，根据这个距离来设置小方块之间的画线的粗细和透明度，给人一种若即若离的感觉。在鼠标交互方面，设计的更为巧妙，通过判断各个小方块和鼠标小方块的距离，当鼠标小方块附近的小方块逃离到一定距离时，便修改小方块的坐标，形成束缚的效果。       
其他细节和具体实现，请看https://github.com/jc1144096387/canvas_nest

## 折叠侧边栏
使用了jquery的animate动画函数，实现折叠，并使用jquery的ajax方法获取和风天气的天气数据，实现简易的天气功能。   
和风天气需要注册，免费用户每天有一定的请求量限制，可以获取3天还是7天的天气数据，我这里只用到了当天的天气数据，有兴趣的同学可以去注册一个。

## 基于模板对头部导航栏进行重构
将头部导航栏独立为.tpl文件，并使用封装jquery的get方法，做成了简单的include函数，在需要使用的页面include即可。

## 基于模板和json数据对文章简介进行重构
将文章简介独立为.tpl文件，并使用占位符（在这里我使用~作为占位符）结合json数据文件来实现简单的模板功能。在首页中，直接导入tpl和json文件来渲染。在文章列表页，我实现了一个简易的分页功能，根据页码导入对应的json文件来渲染。

## 具体文章页底部的评论模块
这个是操作DOM的小demo，一刷新页面，数据就没了。   
监听按钮的点击事件，当点击按钮时，获取表单中的数据，通过创建元素并添加到父元素中，实现添加评论的功能。
